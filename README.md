# Recruvo — AI-Powered Resume Screening Portal

> Find your dream job. Hire top talent. Powered by AI.

🌐 **Live Demo:** [resume-screening-portal-alpha.vercel.app](https://resume-screening-portal-alpha.vercel.app)

---

## 📌 About

Recruvo is a full-stack AI-powered hiring platform that connects job seekers with employers through intelligent resume screening and applicant tracking.

- **Job Seekers** can browse thousands of openings, upload their resume, and apply in one click
- **Employers** can post jobs, view applicants, and let AI rank and screen candidates automatically
- **AI Engine** parses resumes and matches candidates to job descriptions for smarter hiring

---

## ✨ Features

- 🔐 Secure user authentication (login, registration, session management)
- 📄 Resume upload and AI-based parsing
- 💼 Job browsing and one-click apply
- 🏢 Employer dashboard with applicant tracking system (ATS)
- 📊 Smart candidate ranking based on job requirements
- 🌐 Fully deployed and production-ready on Vercel

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Samyuckkk/resume-screening-portal.git

# Navigate to backend
cd resume-screening-portal/backend

# Install dependencies
pip install -r requirements.txt

# Run the server
python app.py
```
